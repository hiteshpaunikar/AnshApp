Ansh Website : The Ansh website is a Progressive Web Application developed by SAC to facilitate and streamline various services for its employees and retired employees. The website aims to enhance productivity and provide essential information within the organization.

* Key Features of the Ansh Website:

  • View Latest Doctors / Hospital / Lab. List
  • Search Doctor's Details
  • Apply filter on doctor's details like Specialisation, Doctor Type, Address etc.
  • View Latest Guest House Information
  • Directly make call from app (To Doctors / To Guest House)
  • View location of Clinic / hospital / Guest House on map
  • Voice based Doctor search / Guest House Search
  • System automatically update the details when new data available
  • Supplementary Items List
  • Inadmissible Items/Medicines List
  • Important Telephone Number List
  • CHSS Entitlement of Ward Charges
  • Holiday List
